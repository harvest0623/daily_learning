# 组件通讯
- 父子组件：将父组件中的变量传递给子组件使用
    1. 父组件直接在子组件的标签上绑定属性值
    2. 子组件通过参数 props 接受  (props 接受到的数据是只读的)

- 子父组件：将子组件中的变量传递给父组件使用
    1. 父组件定义一个接收数据的函数
    2. 父组件将函数通过 props 传递给子组件
    3. 子组件调用该函数并传递数据

- 兄弟组件：
    1. 子组件 1 给父组件传递值
    2. 父组件再传给组件 2
    (仓库)

- 跨组件通信：
    1. 父组件createContext 创建一个上下文对象，并Provider传递出一份数据
    2. 需要接受数据的那个后代组件 用useContext将父组件中的上下文对象使用起来并得到值
